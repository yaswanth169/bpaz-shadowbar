# Quick Start Guide

Get up and running with ShadowBar in under 2 minutes.

## 1. Install ShadowBar

Install the package via pip. Requires Python 3.10+.

```bash
pip install shadowbar
```

## 2. Authenticate

Log in with your Barclays credentials to access managed LLMs.

```bash
sb auth microsoft
```

## 3. Create Your First Agent

Use the CLI to scaffold a new agent.

```bash
sb create my-agent
```

This will create a new directory `my-agent` with a basic agent template.

## 4. Run the Agent

Navigate to the directory and run the agent.

```bash
cd my-agent
python agent.py
```
