## Why ShadowBar?

ShadowBar was built to solve the unique challenges of deploying AI agents within a regulated financial environment.

*   **Security First**: Built with strict trust levels, network isolation, and comprehensive audit logging.
*   **Barclays Integrated**: Native support for Microsoft 365 (Outlook, Calendar) and internal APIs.
*   **Developer Friendly**: A Pythonic API that feels natural to use, with a powerful CLI (`sb`) for management.
*   **Anthropic Optimized**: Deep integration with Claude's tool use and reasoning capabilities.
